SAPI是什么？

SAPI是一个基于Spring的帮助开发人员扫描Restful接口的小工具，只需要一行配置，就可以看到相应的接口以及参数。

SAPI如何测试？

打开了http://127.0.0.1:{your.port}/{your.contextPath}/spai后，在每个接口右侧页面上可以看到一个绿色的三角按钮()。对相关参数进行赋值后点击就可以看到相关的输出结果了。

为何接口扫描出来了，但是测试时提示404？

很可能您使用了@Controller，如果不需要使用Thymeleaf等系列模板，建议使用@RestController进行测试。

如何获取源码?
